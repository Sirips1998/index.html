# # asahi-linktree

หน้าโปรไฟล์ลิงก์สำหรับ `Asahi | ป้ายยา` ซึ่งเป็นหน้า HTML สวยงามสำหรับรวมช่องทางติดต่อและผลงานไว้ในที่เดียว

## รายละเอียด

- ไฟล์หลัก: `asahi-linktree.html`
- เป็นหน้าเว็บแบบ static ที่แสดงผลลิงก์ Instagram, TikTok, LINE และพิกัดการจ้างงาน
- มีสไตล์การออกแบบแบบ dark theme พร้อมอนิเมชันและเอฟเฟกต์ particle และ ripple
- ปรับแก้รูปโปรไฟล์และข้อความได้ง่ายจากโค้ดภายในไฟล์

## การใช้งาน

1. เปิดไฟล์ `asahi-linktree.html` ในเบราว์เซอร์
2. แก้ไขข้อมูลใน `<script>` สำหรับ `TIKTOK_POSTS` หากต้องการอัปเดต TikTok ล่าสุด
3. อัปโหลดไฟล์นี้ไปที่ GitHub Pages หรือโฮสต์ static ใดก็ได้เพื่อใช้งานเป็นหน้า linktree
